# Krisna Media

**SOUND • LIGHTING • STUDIO • GEAR**

Website resmi Krisna Media — platform pemesanan jasa sound system, lighting, studio musik, dan perlengkapan event di Banjarmasin.

## Fitur

- 🛒 **Booking Online** — Pesan layanan langsung dari web, lengkap dengan proteksi double-booking
- 🔐 **Auth Member/Admin** — Daftar sebagai member, admin panel khusus untuk superadmin
- ⭐ **Testimoni** — Member bisa kirim testimoni, admin yang approve
- 📋 **Admin Panel** — Kelola produk, booking, testimoni, dan pesanan
- 📱 **Responsive** — Tampilan optimal di desktop & mobile

## Tech Stack

| Teknologi | Kegunaan |
|-----------|----------|
| Next.js 16 (App Router) | Frontend + Backend |
| TypeScript | Type safety |
| Tailwind CSS v4 | Styling |
| Supabase (PostgreSQL) | Database & Auth |
| Vercel | Hosting |
